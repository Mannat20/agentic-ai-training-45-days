# Flight Management System Assignment

## Objective

The objective of this assignment is to implement real-world data processing operations on flight data using Python.

The assignment focuses on three fundamental operations:

- Searching
- Sorting
- Filtering

These operations are commonly used in applications like:

- MakeMyTrip
- Goibibo
- EaseMyTrip
- Booking.com

---

# Problem Statement

Given a collection of flight records, implement functionalities that allow users to:

1. Search for flights.
2. Sort flights based on different criteria.
3. Filter flights according to user requirements.

# 1. Search Flights

## Objective

Find a particular flight from the available records.

### Search Criteria

- Flight Code

### Learning Outcomes

- Data traversal
- Conditional searching
- Linear Search implementation

---

# 2. Sort Flights

## Objective

Arrange flights according to a specific property.

### Sorting Criteria

- Price
- Duration

### Learning Outcomes

- Comparison-based sorting
- Bubble Sort implementation
- Data organization

---

# 3. Filter Flights

## Objective

Display only those flights that satisfy certain conditions.

### Filter Criteria

- Flights below a certain price
- Flight Fare
- Flight Duration
- Flights of a particular airline

### Learning Outcomes

- Applying conditions on collections
- Extracting subsets of data
- Real-world filtering implementation
