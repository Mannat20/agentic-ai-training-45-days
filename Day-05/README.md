## 📚 Session Overview

Day 05 focused on understanding Python operators and decision-making constructs. We explored different categories of operators including arithmetic, assignment, logical, membership, bitwise, and shift operators. The session also covered conditional statements (`if`, `if-else`, `if-elif-else`, and nested `if`) which are fundamental for controlling program flow.

Towards the end of the session, the trainer explained a practical approach for handling **promo codes** using conditional logic. The assignment for the day involved solving the **Brick Problem**, which emphasized logical thinking and loop implementation.
