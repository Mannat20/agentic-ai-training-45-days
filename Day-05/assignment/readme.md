# Day 05 Assignment

## Problem: Brick Problem

### Objective

Implement a Python program to determine which person places the last brick based on the given brick placement pattern.

---

## Problem Statement

John and Jack are building a wall.

- John starts first.
- In round `i`, John places `i` bricks.
- Jack then places `2 × i` bricks.
- This continues until all bricks are placed.

If the total number of bricks ends while someone is placing bricks, determine:

- Who placed the last brick.
- How many bricks that person placed in their final turn.

---

## Concepts Used

- Variables
- while Loop
- for Loop
- Conditional Statements
- Logical Thinking

---

## Algorithm

1. Read the total number of bricks.
2. Initialize total bricks placed as 0.
3. Start round number from 1.
4. John places `i` bricks.
5. Check after every brick whether the wall is complete.
6. Jack places `2 × i` bricks.
7. Continue until all bricks are placed.
8. Display the person who placed the last brick and the number of bricks placed in their final turn.
