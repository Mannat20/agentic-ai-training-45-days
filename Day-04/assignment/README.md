# Day 04 Assignment - Chess Board Pattern

## 🎯 Objective

The objective of this assignment was to understand multidimensional data representation using nested loops by creating a Chess Board pattern.

---

# 📝 Problem Statement

Design a Chess Board structure using Python.

The board should represent alternating black and white squares using logical conditions and nested loops.

---

# 💡 Approach

The solution uses nested loops.

- Outer loop represents rows.
- Inner loop represents columns.
- A condition determines whether a square should be black or white based on row and column indices.
