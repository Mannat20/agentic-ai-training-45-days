## Concepts Learned

### Function in Class
Methods are functions defined inside a class that operate on the object's data.

### Returning Data using Dictionary
Objects can return their information in dictionary format, making the data easier to display, process, or convert to JSON.

### Circular Doubly Linked List (CDLL)
A linked list where:

- Every node has a `next` reference.
- Every node has a `previous` reference.
- Last node points to the first node.
- First node points to the last node.
